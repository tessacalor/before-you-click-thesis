# before-you-click-thesis
This repository contains the analysis code for the master's thesis "Before You Click: Visual Framing in YouTube News Thumbnails and Video Content" (University of Amsterdam, 2026). The notebooks implement a computational pipeline for analyzing YouTube news thumbnails covering the Venezuela–United States conflict, including CLIP embeddings, UMAP dimensionality reduction, color analysis, text overlay detection via EasyOCR, and CLIP-based affect scoring. The analysis was conducted in Google Colab with data stored in Google Drive. The accompanying dataset is available on Zenodo at https://doi.org/10.5281/zenodo.20753854.
